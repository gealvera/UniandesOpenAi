Este sitio contendrá los talleres del curso Desarrollador de soluciones de Inteligencia Artificial (Azure Open AI)

Sumérgete en el futuro de la inteligencia artificial con nuestro revolucionario curso Desarrollador de soluciones de Inteligencia Artificial (Azure Open AI). ¡Aprende a utilizar los servicios generativos de Microsoft y adquiere habilidades prácticas para crear soluciones de IA increíbles!

🌟 BIENVENIDOS 🌟

Aquí podrás encontrar los talleres que formarán parte de nuestro viaje a lo largo del curso.

Indice

  • Contextualización

    • 1. Configurando la nube de Azure.
      Propósito: En esta sesión inicial, vamos a abordar la configuración de los componentes en la plataforma de nube en Azure, los cuales desempeñarán un papel fundamental a lo largo de nuestro curso

    • 2. Preparando el entorno.
      Propósito: En esta sesión, vamos a abordar la configuración del entorno para poder hacer uso de las Azure Functions desde Visual Studio Code.

    • 3. Concepción de la Function (Creación y despliegue)..
      Propósito: En esta sesión, nos enfocaremos en la comprensión y dominio de los conceptos fundamentales para la creación, y despliegue de funciones en Azure. Estos conocimientos serán esenciales para lograr una ejecución eficaz y exitosa en el entorno de desarrollo en la nube de Azure.

      • 3.1 Creación Function en VSC (Paso a Paso).
        Propósito: En esta sesión, se va a seguir el paso a paso para crear una Function desde Visual Studio Code.

      • 3.2 Despliegue Function desde VSC (Paso a Paso).
        Propósito: En esta sesión, se va a seguir el paso a paso para desplegar una Function desde Visual Studio Code al recurso del Portal.

  • Talleres

    • En clase

      • 1. Taller 1.1 - Ejercicio básico de conexion en Azure Functions y OpenAI Completion.
        Propósito: En este taller se va a realizar la configuración con el API de OpenAI y describir sus diferentes parámetros al momento de consumir el API y se va a implementar una función para llamar el método de open AI para consumir con un prompt

      • 2. Taller 1.2.a - Ejercicio básico de conexion en Azure Functions, OpenAI Completion con Embeddings.
        Propósito: En este taller se va a realizar la integracion entre Azure Functions con el API de OpenAI y utilizar la estrategia de Embeddings como fuente de conocimiento para posteriormente pasar ese contenido como fuente al prompt y generar la respuestas enfocadas al contexto previo

      • 2. Taller 1.2.b - Ejercicio básico de conexion en Azure Functions, OpenAI Completion con Embeddings (Lectura de PDF).
        Propósito: En este taller se va a realizar la integracion entre Azure Functions con el API de OpenAI y utilizar la estrategia de Embeddings como fuente de conocimiento para posteriormente pasar ese contenido a la base de datos Cosmos y al servicio de Cognitive Search (Búsqueda IA)

      • 3. Taller 1.3 - Ejercicio básico de conexión a Cosmos DB desde Azure Functions.
        Propósito: En este taller se va a realizar la conexión desde las Azure Functions al recurso de Cosmos DB creado previamente en el portal de Azure. La propuesta para este taller es establecer la conexión y poder obtener y crear registros en la base de datos.

      • 4. Taller 1.4 - Ejercicio básico de conexión a Cognitive Search (Búsqueda IA) desde Azure Functions.
        Propósito: En este taller se va a realizar la conexión desde las Azure Functions al recurso de Cognitive Search (Búsqueda IA) creado previamente en el portal de Azure. La propuesta para este taller es establecer la conexión y poder obtener contextos generados en Cognitive Search (Búsqueda IA) apartir de una pregunta del usuario.

      • 5. Taller 1.5 - Ejercicio básico de generación de contexto mediante Cognitive Search (Búsqueda IA) y respuesta con Open AI.
        Propósito: En este taller se va a realizar la unión de los talleres anteriores, en el cual se va a generar un contexto haciendo uso del recurso de Cognitive Search (Búsqueda IA), y luego con ese contexto y la pregunta se hará la petición para que Open AI.

      • 6. Taller 2.1 - Ejercicio práctico Prompts, Parsers y Chat conversacional usando LangChain.
        Propósito: En este taller se va a realizar la parte práctica de los conceptos vistos en la parte inicial de la clase. Se hará uso el framework de LangChain para poner en practica los conceptos del chat conversacional y la memoria.

  • Proyectos por entregar

    • 1. Proyecto: Constitución Colombiana - Parte 1 - Generación de Data.
      Propósito: En la primera parte del proyecto se va utilizar el conocimiento dado en el taller de clase 1.1, 1.2, 1.3.
